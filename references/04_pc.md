## 공통 점검 항목

### PC-01. 화면 보호기 및 자동 잠금

- **위험도**: 중
- **점검 내용**: 일정 시간 후 화면 잠금 설정 여부
- **양호 기준**: 5~10분 이내 자동 잠금, 재개 시 패스워드 필요
- **취약 기준**: 화면 보호기 미설정 또는 재개 시 패스워드 불필요

### PC-02. 소프트웨어 업데이트

- **위험도**: 상
- **점검 내용**: OS 및 주요 소프트웨어 보안 패치 최신화 여부
- **양호 기준**: OS 및 주요 앱(브라우저, Office 등) 최신 보안 패치 적용
- **취약 기준**: 6개월 이상 보안 패치 미적용

### PC-03. 바이러스 백신 설치 및 업데이트

- **위험도**: 상
- **점검 내용**: 백신 소프트웨어 설치 및 최신 시그니처 업데이트 여부
- **양호 기준**: 백신 설치, 실시간 보호 활성화, 24시간 이내 시그니처 업데이트
- **취약 기준**: 백신 미설치 또는 시그니처 장기간 미업데이트

### PC-04. 공용 Wi-Fi 사용 제한

- **위험도**: 중
- **점검 내용**: 비인가 또는 공용 Wi-Fi 사용 시 VPN 사용 여부
- **양호 기준**: 공용 Wi-Fi 사용 시 반드시 VPN 사용
- **취약 기준**: 공용 Wi-Fi를 VPN 없이 사용

## Windows PC

### WPC-01. 패스워드 정책

- **위험도**: 상
- **점검 방법** (PowerShell):

  ```powershell
  net accounts
  ```

- **양호 기준**: 최소 8자, 복잡성 조건 사용, 최대 사용 기간 90일
- **취약 기준**: 패스워드 정책 미적용

### WPC-02. BitLocker 전체 디스크 암호화

- **위험도**: 상
- **점검 내용**: 로컬 디스크 암호화 설정 여부
- **점검 방법** (PowerShell):

  ```powershell
  manage-bde -status
  Get-BitLockerVolume
  ```

- **양호 기준**: BitLocker 활성화 (ProtectionStatus: On)
- **취약 기준**: BitLocker 미사용 (분실·도난 시 데이터 노출)

### WPC-03. Windows Firewall 활성화

- **위험도**: 상
- **점검 방법** (PowerShell):

  ```powershell
  Get-NetFirewallProfile | Select-Object Name, Enabled
  ```

- **양호 기준**: 모든 프로파일 Enabled
- **취약 기준**: 방화벽 비활성화

### WPC-04. UAC (사용자 계정 컨트롤) 활성화

- **위험도**: 중
- **점검 방법** (PowerShell):

  ```powershell
  Get-ItemProperty "HKLM:\SOFTWARE\Microsoft\Windows\CurrentVersion\Policies\System" -Name EnableLUA
  ```

- **양호 기준**: `EnableLUA = 1` (UAC 활성화)
- **취약 기준**: UAC 비활성화

### WPC-05. 자동 실행(AutoRun) 비활성화

- **위험도**: 중
- **점검 내용**: USB·CD 등 외부 매체 자동 실행 비활성화
- **점검 방법** (PowerShell):

  ```powershell
  Get-ItemProperty "HKLM:\SOFTWARE\Microsoft\Windows\CurrentVersion\Policies\Explorer" -Name NoDriveTypeAutoRun
  ```

- **양호 기준**: NoDriveTypeAutoRun = 255 (모든 드라이브 AutoRun 비활성화)
- **취약 기준**: AutoRun 활성화

## macOS

### MAC-01. FileVault 전체 디스크 암호화

- **위험도**: 상
- **점검 방법**:

  ```bash
  fdesetup status
  ```

- **양호 기준**: `FileVault is On.`
- **취약 기준**: FileVault 비활성화

### MAC-02. 화면 잠금 설정

- **위험도**: 중
- **점검 방법**:

  ```bash
  defaults read com.apple.screensaver askForPassword
  defaults read com.apple.screensaver askForPasswordDelay
  ```

- **양호 기준**: `askForPassword=1`, `askForPasswordDelay=0`
- **취약 기준**: 화면 잠금 해제 시 패스워드 불필요

### MAC-03. 방화벽 활성화

- **위험도**: 상
- **점검 방법**:

  ```bash
  /usr/libexec/ApplicationFirewall/socketfilterfw --getglobalstate
  ```

- **양호 기준**: `Firewall is enabled.`
- **취약 기준**: 방화벽 비활성화
- **조치 방법**: 시스템 환경설정 → 보안 및 개인 정보 → 방화벽 → 켜기

### MAC-04. 게이트키퍼 활성화

- **위험도**: 중
- **점검 내용**: App Store 또는 서명된 앱만 실행 허용 여부
- **점검 방법**:

  ```bash
  spctl --status
  ```

- **양호 기준**: `assessments enabled`
- **취약 기준**: `assessments disabled` (모든 앱 실행 허용)

## Linux PC

### LPC-01. 전체 디스크 암호화 (LUKS)

- **위험도**: 상
- **점검 내용**: LUKS를 통한 디스크 암호화 설정 여부
- **점검 방법**:

  ```bash
  lsblk -o NAME,FSTYPE | grep crypto_LUKS
  cryptsetup status /dev/mapper/luks-*
  ```

- **양호 기준**: LUKS 암호화 디스크 사용
- **취약 기준**: 암호화 미적용

### LPC-02. 방화벽 설정

- **위험도**: 중
- **점검 방법**:

  ```bash
  # Ubuntu/Debian
  ufw status
  # CentOS/RHEL
  firewall-cmd --state
  ```

- **양호 기준**: 방화벽 활성화, 불필요한 포트 차단
- **취약 기준**: 방화벽 비활성화

### LPC-03. SSH 불필요 시 비활성화

- **위험도**: 중
- **점검 내용**: PC에서 SSH 서버 실행 여부
- **점검 방법**:

  ```bash
  systemctl is-active sshd
  ss -tlnp | grep 22
  ```

- **양호 기준**: SSH 서버 불필요 시 비활성화
- **취약 기준**: 불필요한 SSH 서버 실행 중
